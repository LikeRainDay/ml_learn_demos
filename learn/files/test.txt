为了建立一支支持AI的员工队伍，Microsoft宣布了Intelligent Cloud Hub，该中心已启动，旨在为下一代学生提供支持AI的技能。作为一项为期三年的协作计划，Intelligent Cloud Hub将为约100个机构提供AI基础设施，课程内容和课程，开发人员支持，开发工具，并为学生提供访问云和AI服务的支持。作为该计划的一部分，这家希望扩大业务范围并计划在印度建立强大的开发人员生态系统的雷德蒙德巨人将为所选园区建立核心AI基础设施和IoT中心。该公司将提供AI开发工具和Azure AI服务，例如Microsoft Cognitive Services，Bot Services和Azure Machine Learning.Manish Prakash 微软印度公司卫生与教育部总经理-PS：“随着AI成为当今时代的决定性技术，它正在改变着人们的生活和行业，明天的工作将需要不同的技能。这将需要更多的合作和培训因此，对于教育机构来说，整合新的云技术和AI技术变得比以往任何时候都更为重要。该计划旨在扩大机构设置并在教育工作者之间建立能力，以教育未来的劳动力”。该计划旨在建立认知技能，并深入了解为整个行业的应用程序开发智能云连接解决方​​案。今年四月初，该公司宣布将“ Microsoft AI专业计划”作为对公众开放的学习之路。开发该程序的目的是为希望通过一系列在线课程来磨练其AI和数据科学技能的程序员提供准备工作的技能，这些课程还包括动手实验和专家指导。该计划还包括以开发人员为中心的AI学校，该学校提供了大量资产来帮助培养AI技能。
